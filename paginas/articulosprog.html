Primeros pasos sobre Python

Programación

4 de enero 2025

Primero de todo, tenemos que elegir un editor de texto plano. En mi caso, yo uso Visual Studio Code. Python es el lenguaje de programación más fácil para empezar, así que aquí dejo lo más básico de él.
La extensión del archivo python es .py, y es necesario descargar el lenguaje, desde su página oficial. Una vez instalado, ya podremos empezar a programar. Lo más común es empezar creando un archivo que muestre por la pantalla "Hola mundo". Para hacer esto en python, simplemente es con la línea print("Hola mundo") . Ahora, ejecutamos el archivo python (sin tener que compilarlo) o desde el propio Visual Studio, arriba a la derecha, donde sale para ejecutarlo.
print("") sirve para mostrar texto por pantalla. Las variables sirven para almacenar un texto, número, etc. También podemos usarlo para preguntarle al usuario algo y que lo almacene, ya sea para mostrarlo por la pantalla, sumarlo a algo, etc. En este caso, podríamos hacerlo con variable1=input("Introduce la variable: ") ; tendremos que introducir un valor y se guardará en la variable variable1.

Hay varios tipos de datos: numéricos (int), texto (string), bool (verdadero/falso), float para números con decimales. Para sumar, restar, multiplicar y dividir números se hace con +, -, * y /.

Primeros pasos sobre C#

Programación

19 de marzo 2025

C# es un lenguaje de programación orientado a objetos. Esto significa que está diseñado para basar el código en objetos y clases, para una mejor organización.
Lo primero de todo es instalar C#, desde la página oficial. La instalación cambia si lo instalamos en Windows o en alguna distribución de Linux. En linux, tenemos que ejecutar apt install mono-complete y darle permisos con chmod +x. Después, ejecutamos msc -out:fichero.exe fichero.cs. Lo ejecutamos con mono fichero.exe. Ya estaría instalado.
Yo voy a usar Visual Studio Code. Los ficheros c# tienen la extensión .cs . La estructura básica sería la siguiente:
using System;
class HelloWorld
{
.....static void Main(string[] args)
.....{
..........string mistring = "Hola mundo";
..........Console.WriteLine(mistring);
.....}
}
Con esto, guardaríamos una variable que contiene "Hola mundo" y la mostraríamos por pantalla. WriteLine sirve para mostrar una variable o un tipo de dato por pantalla. Luego tendríamos que recompilarlo para ejecutarlo y transformarlo en un fichero.exe.
En windows, tendrías que poner la ruta entera donde está descargado C# y usar csc archivo.cs. Recomiendo bastante guardar la ruta de C# en una variable de windows para usarlo más facilmente.
