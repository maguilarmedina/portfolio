Instalando Arch Linux

Sistemas operativos

2 de febrero 2025

Muchos dicen que instalar Arch Linux es muy complicado, pero con tener mínimos conocimientos de informática, se vuelve una tarea muy fácil. Aunque no quita que al principio sea muy confuso.
Primero hay que instalar la imagen desde la página oficial. Yo voy a instalarlo en VirtualBox. Para más comodidad, lo voy a usar con una red puente, para poder instalarlo con ssh. Una vez cargado y la interfaz de comandos, usamos service start sshd y podremos conectarnos a su ip (ip addr show para ver su ip).
Luego, ejecutamos el comando archinstall y nos saldrá un menú con el que podremos elegir, de forma simplificada, las características de la instalación, como el nombre, el entorno gráfico (yo elijo GNOME), particiones, etc. Una vez terminado de configurar y se haya instalado, tendremos que reiniciar con exit, umount -a y reboot.
Una vez instalado, puede que la interfaz estén caídas ip addr show, sudo ip addr add ip/máscara nombredeinterfaz, sudo ip link set nombreinterfaz up/down (up para activarla) y sudo ip route add default via gateway dev nombreinterfaz.
Para instalar un paquete, en vez de usar apt como se haría normalmente en Linux, se hace con sudo pacman -S .
